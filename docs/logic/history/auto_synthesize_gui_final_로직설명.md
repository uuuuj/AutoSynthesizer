# `auto_synthesize_gui_final.py` 로직 설명 (2026-03-18)

## 목적

**엑셀 데이터를 입력받아 합성(가짜) 데이터를 자동 생성하는 GUI 도구**입니다. 원본 데이터의 통계적 특성(분포, 상관관계, 제약조건)을 보존하면서 개인정보/기밀 데이터를 가짜값으로 대체합니다.

---

## 전체 파이프라인 (5단계)

| 단계 | 설명 |
|------|------|
| **① 파일 선택** | 엑셀 파일 로드 (xlwings 우선, openpyxl 폴백) |
| **② 파일 분석** | 500행 초과 시 상위 500행만 유지 + 컬럼별 타입 자동 감지 (수치/날짜/범주) |
| **③ 확인/수정** | 컬럼명 변경 + 문자열 컬럼의 원본값→가짜값 매핑 입력 |
| **④ 변환 확정** | 매핑 중복 검증 후 변환 계획 확정 |
| **⑤ 합성 실행** | 8단계 합성 파이프라인 수행 |

## 핵심 로직 상세

### 1. 엑셀 로드 (`load_excel`, line 398)

- `xlwings`(COM 기반) → `openpyxl`(순수 파이썬) 순으로 시도
- 로드 후 `_postprocess_dataframe`에서 한글 날짜(`2024년 3월 15일`), 오전/오후 시간, 콤마 숫자 등을 자동 파싱

### 2. 컬럼 타입 감지 (`auto_detect_column_type`, line 416)

- `datetime`: datetime64 타입이거나 날짜 패턴 문자열
- `numerical`: 숫자형이면서 고유값 비율 > 5%
- `categorical`: 그 외 전부 (문자열, 저카디널리티 숫자 포함)

### 3. 문자열 합성 (`synthesize_text_columns`, line 441)

- 사용자가 GUI에서 입력한 1:1 매핑(`원본값→가짜값`)으로 범주형 문자열을 치환
- null 값은 그대로 보존

### 4. 가짜 데이터 자동 생성 (line 87~115)

- 한글 이름 패턴 → `generate_fake_persons` (성+이름 조합)
- 회사명 컬럼 → `generate_fake_companies` (Alpha-Corp 형태)
- 그 외 → `generate_auto_codes` (컬럼명 기반 코드)
- ID형 컬럼은 `S0001, S0002...` 형태로 자동 재생성

### 5. 수치/날짜 합성 (`generate_numeric_datetime`, line 509)

- **Gaussian Copula** 방식으로 수치 컬럼 간 상관관계를 보존
  - 원본 데이터를 rank → 정규분포 변환 → 공분산 행렬 추정 → 다변량 정규분포 샘플링 → 역변환
- 날짜 컬럼은 원본 분포에서 랜덤 샘플링, 시작-종료 쌍은 기간 분포를 유지
- **제약조건 자동 감지**: 양수 제약, 0~100 범위, 날짜 순서(시작 < 종료)
- null 비율을 원본과 동일하게 유지

### 6. 추가 품질 보장 로직

- **함수 종속성** (line 659): A 컬럼 값이 B 컬럼 값을 결정하는 관계를 감지하고 합성 후 복원
- **가중 샘플링** (line 1783): 범주형 컬럼의 빈도 분포를 보존하여 행 조합
- **ID 유니크성** (line 1839): 원본에서 고유했던 ID 컬럼은 합성 데이터에서도 고유 ID 재생성
- **매핑 중복 검증** (line 1008): 같은 가짜값이 다른 원본값에 매핑되면 경고 (복원 불가 방지)
- **품질 검증** (`validate_quality`, line 678): KS 검정(수치/날짜) + 빈도 분포 비교(범주)로 0~100% 품질 점수 산출

### 7. 출력 파일 (4종)

| 파일 | 설명 |
|------|------|
| `.xlsx` | 합성 데이터 |
| `_변환키.json` | 컬럼명 변경 + 값 매핑 (복원용 키) |
| `_description.json` | 합성 방법 설명 |
| `_품질리포트.json` | 품질 점수, 상관관계, 제약조건 정보 |

### 8. 복원 탭 (탭2, line 1260)

- 합성 데이터 `.xlsx` + `_변환키.json`을 입력하면
- 가짜값→원본값 역매핑 + 변경된 컬럼명→원본 컬럼명으로 복원하여 `_복원.xlsx`로 저장
